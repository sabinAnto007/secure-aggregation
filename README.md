# Secure Aggregation Benchmark

Code: `secure_aggregation.py`  
Datasets: `heart_disease_clean.csv`, `harmergedd.csv`, `faostattt_clean.csv`

Scope: Compare computational time and communication bytes across Masking, AHE, FE on Heart, HAR, FAOSTAT at small/moderate/full cohorts.

Reproduce:
1) Install Python 3.x + requirements
2) Run `python secure_aggregation.py` and follow the prompts
3) Results CSVs will be created; plots can be generated from them
